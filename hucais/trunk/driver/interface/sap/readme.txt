=================== 说明 ==============================
前提：基于python 2.7

=================== 安装 ==============================
1、windows下:
    > 在C盘下建立文件夹： c:/saplib
    > 将lib文件夹下的librfc32.dll拷贝到c:/saplib下
    > 将c:/saplib增加到系统变量PATH下
    
    > 将pysaprfc_1_0_0.zip解压到某个地方，如 d:/pysaprfc
    > 执行cmd命令，进入命令行，切换到d:/pysaprfc目录
    > 执行 python setup.py install 命令
